# Quick Start Guide

## 🚀 Get AuraPlay RGB

### Windows

#### For Regular Users (Easiest)
1. Download **AuraPlay-RGB-Setup.exe** from releases
2. Double-click to run installer
3. Choose installation folder
4. Done! Desktop shortcut created

#### For Advanced Users (Portable)
1. Download **AuraPlay RGB.exe** from releases
2. Save anywhere you want
3. Double-click to run
4. No installation needed

### Mobile (iOS & Android)

1. **Scan QR Code** below or visit: **https://auraplay-rgb.vercel.app**

   ```
   [QR Code Here]
   ```

2. **iOS**: 
   - Open in Safari
   - Tap Share → Add to Home Screen
   - Tap "Add"

3. **Android**:
   - Open in Chrome
   - Tap ⋮ → Install app
   - Tap "Install"

## 🔐 First Time Setup

### Get Your Spotify Credentials (Free)

1. Go to: https://developer.spotify.com/dashboard
2. Sign in with Spotify (create account if needed)
3. Accept terms & continue
4. Click "Create an App"
5. Name: "AuraPlay RGB"
6. Check boxes & agree
7. Copy your **Client ID**
8. Copy your **Client Secret**

### Configure AuraPlay

**Windows App:**
- First launch will ask for credentials
- Paste Client ID
- Paste Client Secret
- Click "Save"
- Done!

**Web Version:**
- Tap ⚙️ (Settings)
- Paste Client ID
- Paste Client Secret
- Tap "Save"
- Done!

## ⚠️ Troubleshooting

### "Premium Required" Error
- Your Spotify account is Free
- Premium users can control playback
- Free users can view current song
- Upgrade to Premium if you want playback control

### "No Song Playing"
- Make sure Spotify is open somewhere
- Play a song first
- AuraPlay will show it

### Connection Issues
- Check internet connection
- Restart AuraPlay
- Make sure Spotify is running

### App Won't Start
- Windows: Make sure you have admin rights
- Mobile: Try refreshing the page
- Check your internet connection
