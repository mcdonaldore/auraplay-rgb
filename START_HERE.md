## **OPTION 1: Windows Distribution (5 minutes)**

### Go to GitHub.com and:
1. **Sign up** (if you don't have account)
2. Click **+** → **New repository**
3. Name it: `auraplay-rgb`
4. Click **Create repository**
5. You'll see: `https://github.com/YOUR_USERNAME/auraplay-rgb.git`

### Open PowerShell in your project folder and run:

```powershell
cd "c:\Users\SATURO\OneDrive\Documents\Auraplay-RGB"

git config --global user.name "Your Name"
git config --global user.email "your@email.com"

git init
git add .
git commit -m "v1.0.0"
git remote add origin https://github.com/YOUR_USERNAME/auraplay-rgb.git
git branch -M main
git push -u origin main
```

When asked for password: Use GitHub Personal Access Token
- Go to: https://github.com/settings/tokens
- Click "Generate new token (classic)"
- Check `repo` box
- Click "Generate token"
- Copy & paste it

### Create Release:
1. Go to your repo on GitHub
2. Click **Releases**
3. Click **Create a new release**
4. Tag: `v1.0.0`
5. Upload: `dist/AuraPlay RGB.exe`
6. Click **Publish**

### Windows Download Link:
```
https://github.com/YOUR_USERNAME/auraplay-rgb/releases
```

## **OPTION 2: Mobile Distribution (3 minutes)**

### Install Node.js:
- Go to: https://nodejs.org/
- Download & install LTS

### Deploy to Vercel (Free):
```powershell
npm install -g vercel

cd "c:\Users\SATURO\OneDrive\Documents\Auraplay-RGB\web_version"
vercel
```

Follow prompts → You'll get URL like: `https://auraplay-rgb.vercel.app`

### Mobile Download Link:
```
https://auraplay-rgb.vercel.app
```

Users: Open in Safari/Chrome → Share → Add to Home Screen
